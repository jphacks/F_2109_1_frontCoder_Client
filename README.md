- PC

## 使用言語

- TypeScript + React

## 使用ライブラリ / API / サービス

- Material ui
- Firebase Auth
- Firebase FireStore
- CodeSandBox API

## 概要

フロントエンドは Amazon Amplify を使用してアップロードしています。CI/CD を利用しており、push されると自動でサーバーに後悔することができます。
